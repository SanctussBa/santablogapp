About this blog site

This site is created for educational purposes. It is simple blog web site for anyone to use.

Main features are:

>>> Registration with e-mail and password
>>> Login and personal Account creation
>>> Writing blogs (Creation of Blog Title, Date when blog is created and Blog Content - text only)
>>> Updating Blog title, content and Profile picture
>>> Deleting posts
>>> Possibility to see other Blog User's only blogs.

Used tools:

[HTML5] and [CSS3 with SASS] and Google Fonts
[JavaScript] just a snippet for modal show up
[Python3] as main Back-End language
[Flask] - Python web application framework.
